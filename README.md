# sv-bedrock
Scott's Wordpress Site Custom Starting Point

## Prerequisites
- [MAMP](https://www.mamp.info/en/downloads/)
- Node
- NPM
- [Composer](https://getcomposer.org/download/)

Copy `.env.example` file to `.env`

## Project Root
`composer install`

## Theme Folder `src/app/sv-theme`
`npm install`
`npm run build`
`npm run watch`
